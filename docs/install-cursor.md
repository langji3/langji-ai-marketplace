# Install AI Dev Protocol In Cursor

1. Open `plugins/ai-dev-protocol/adapters/cursor/ai-dev-protocol.mdc`.
2. Copy it into the target project's `.cursor/rules/`.
3. Reopen the target project in Cursor if needed.
4. Verify the rule is available to the assistant in that workspace.

## Notes

- Cursor uses the distributed adapter file from this marketplace snapshot.
- The source-of-truth for the rule content remains `ai-dev-protocol`.
